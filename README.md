# ASCII Art 

This project demonstrates creating **live ASCII art from your webcam** using Python. It includes **three versions**:

1. **ASCII TERMINAL** – ASCII output printed in the terminal.  
2. **ASCII ART** – ASCII displayed in a separate OpenCV window in grayscale.  
3. **Colored ASCII** – ASCII displayed in a separate OpenCV window with colors matching the original camera feed.

---

## Features

- Capture live video from a webcam
- Convert video frames into ASCII characters
- Display in terminal or OpenCV window
- Optional colored output for a more realistic effect
- Adjustable ASCII resolution for performance optimization

---

- Python 3.x
- OpenCV (`cv2`)
